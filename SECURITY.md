# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Inboxes, please report it responsibly.

**Email:** [security@inboxes.net](mailto:security@inboxes.net)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

**Do not** open a public GitHub issue for security vulnerabilities.

## Scope

This policy covers the Inboxes codebase (backend and frontend). Infrastructure and hosting issues specific to inboxes.net should also be reported to the email above.
